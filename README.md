# 🚀 PlanIT — AI Daily Planner for BTech Students

An AI-powered daily planner that generates personalized roadmaps for BTech students based on their goals, year, and availability.

## 🔗 Live App
[Try PlanIT here](https://huggingface.co/spaces/adi-halder/planit)

## 🛠 Built With
- Python
- Groq API (LLaMA 3.3 70B)
- Gradio

## 💡 Features
- Personalized daily plans for any BTech goal
- Step-by-step guidance with Why, How and Tomorrow
- Supports internship prep, placements, studies and skills
